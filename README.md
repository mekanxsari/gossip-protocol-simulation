# Gossip Protocol Simulation

## What is Gossip Protocol?

Gossip protocol is a probabilistic communication algorithm used in distributed systems. Each node randomly selects neighbors, exchanges state information, updates its local state, and repeats this process in rounds. Eventually, all nodes converge to the same information - a property known as eventual consistency.

## Project Features

- Round-based message propagation
- Configurable node count and fanout
- Dead node simulation
- Slow node simulation
- Real-time informed node tracking
- Message counting
- Convergence detection
- Interactive node visualization
